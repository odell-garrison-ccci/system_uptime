# Copilot Test Script Documentation

## Description
This script, `copilot_test.py`, is designed to automate the testing of the Copilot application, simulating user interactions, and verifying the functionality of various features.

## Installation
To install the required dependencies to run the script, execute the following command:

```bash
pip install -r requirements.txt
```

## Usage
Run the script using the following command:

```bash
python copilot_test.py
```

Make sure to change directory to where the script is located or provide the full path to the script file.

## Error Handling
The script includes error handling mechanisms to gracefully manage exceptions that may occur during execution. Common exceptions are caught and logged accordingly.

## Logging
Logging is implemented using Python's built-in `logging` module. Logs will be written to `copilot_test.log` and will include timestamps, log levels, and messages for easier debugging.
